# Fertilizer-Deficiency-Monitoring
"An AI-powered agricultural system that detects nutrient deficiencies in plant leaves using Deep Learning. Features include real-time fertilizer recommendations with pricing, a multi-lingual AI chatbot (4 languages), and an intelligent fertilizer calculator based on crop type, area, and deficiency severity."
AI-Based Fertilizer Deficiency Monitoring System
A robust AI-driven solution designed to detect nutrient deficiencies in crops using computer vision and machine learning.

* Project Overview
This project utilizes TensorFlow, PyTorch, and OpenCV to analyze leaf images and identify specific nutrient deficiencies. The goal is to provide farmers with actionable insights to optimize fertilizer usage.

* Tech Stack
Language: Python
Frameworks: Django (Backend), TensorFlow/PyTorch (Model)
Computer Vision: OpenCV
Database: SQLite/PostgreSQL

* Status
Current Phase: Model training and fine-tuning.
Goal: Improving detection accuracy for multi-nutrient deficiency.

* Features
Leaf Image Scanning:** Detects deficiencies using Computer Vision.
Real-time Recommendations: Suggests fertilizers based on deficiency severity.
Multi-lingual AI Chatbot: Support for 4 languages (Urdu, English, etc.) for farmer guidance.
Intelligent Fertilizer Calculator: Calculates exact dosage based on crop type and area.
Expense Tracker: Helps farmers keep track of their spending.
